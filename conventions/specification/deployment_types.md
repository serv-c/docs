# Deployment Types

The structure is as follows,

```
- <category> (short hand)
  - <instance of category>
```

Examples instances and categories are below,

- downloader (dl)
  - deluge
  - sabnzbd
- indexer (idx)
  - hyra2
  - jackett
- narr (nr)
  - sonarr
  - radarr
- media (md)
  - plex
  - jellyfin
- request (rqt)
  - overseer
